Getting Started

Ushahidi Platform is humanitarian open source software for information collection, visualization and interactive mapping. It is used to build tools for democratizing information, increasing transparency and lowering the barriers for individuals to share their stories. For this project we have integrated a heatmap plugin into the Ushahidi platform. The heatmap plugin was taken from David Kobia's heatmap plugin. The heatmap plugin is used to display all active reports from Ushahidi onto a map. It displays its own heatmap on a different page and shows all the reports as a whole, without the user capability of changing the view/category. For our project, we integrated the heatmap into the main ushahidi map, thus, giving users a chance to switch the view/categories and toggle the heatmap. In addition, we allow the admin a choice to enable or disable the heatmap in the admin dashboard.

What the project is useful

During crisis situations this project will provide critical information to identify areas of great concern. Through Ushahidi, local observers can submit incidents using their mobile phones or the internet, while simultaneously creating a heatmap of events as they happen. Using the heatmap is simple, users can toggle between the original reports layer on/off, depending on their preference. The heatmap can be used as an additional information cue for people to display geospatial archive of events. User can select different categories to display a heatmap of the incidents.
